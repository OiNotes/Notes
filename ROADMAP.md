# Oi/Notes Implementation Plan

## Фаза 1 – Контентная основа
- Описать типы данных (TypeScript + Zod) для людей, артефактов, медиа.
- Перенести существующие заметки в новую структуру `content/people/<slug>.mdx`.
- Настроить генераторы путей и `generateStaticParams` для новых страниц.

## Фаза 2 – Главная сцена
- Сверстать hero с Sora-видео (Intersection Observer, autoplay без звука).
- Создать сетку карточек каталога + фильтры/сортировку.
- Добавить ленивую подгрузку видео-постеров.

## Фаза 3 – Страница героя
- Базовый layout (header, навигация, pлеер, артефакты, «ещё по теме»).
- Реализовать кастомный видеоплеер в стиле Apple Music.
- Подключить динамическую таблицу содержания/scroll-spy.

## Фаза 4 – Артефакты & чтение
- Развернуть «артефакт» секции в формате аккордеонов/полноэкранной читалки.
- Анимации для раскрытия, прогресс чтения, ссылки на Notebook LM PDF.
- Локальный поиск по содержимому (optional).

## Фаза 5 – Визуальные детали
- Полировка темизации (тени, стекло, типографика).
- Добавить микроанимации hover/transition для карточек и контролов.
- Настроить глобальные intro-видео (mute/unmute, переключение).

## Фаза 6 – Производительность
- Lazy loading тяжёлых ассетов, prefetch маршрутов.
- Lighthouse/LCP тюнинг, оптимизация размеров видео и постеров.
- Написать smoke-тесты (visual regression, ESLint, TS).
