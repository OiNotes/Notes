# Музыкальный контент

Здесь хранятся JSON файлы с данными о песнях.

## Структура файла

```json
{
  "id": "unique-id",
  "slug": "url-slug",
  "title": "Название песни",
  "artist": "Исполнитель",
  "audioUrl": "/music/songs/audio.mp3",
  "coverImageUrl": "/music/covers/cover.jpg",
  "duration": 240,
  "description": "Описание перевода",
  "accent": "#d4af37",
  "tags": ["тег1", "тег2"],
  "publishedAt": "2025-01-15T12:00:00Z",
  "lyrics": [
    {
      "id": "line-1",
      "text": "Текст строки на русском",
      "startTime": 1000
    }
  ]
}
```

## Как добавить песню

1. Загрузите MP3 файл в `/public/music/songs/`
2. Загрузите обложку в `/public/music/covers/`
3. Откройте админ-панель (5 кликов на логотип, пароль: 1234)
4. Заполните форму с информацией о песне
5. Используйте лабораторию таймингов для расстановки
6. Сохраните - файл появится здесь автоматически!
